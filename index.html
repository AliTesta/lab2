Definitions:

Table: They are made by using the tags <table> </table> with <tr> </tr> pairs nested inside for each row. Additionally, there are <td> </td> pairs nested in the <tr> </tr> pairs to distinguish cells/columns. However, in the first <tr> </tr> pair, <th> </th> pairs would be used instead inside to create header cells. You can also use <thead> </thead> tags to create the header of a table and <tfoot> </tfoot> tags to create the footer.<caption> </caption> allows to the coder to place a caption at the top on the table.

Colspan vs Rowspan: colspan is used to combine cells across columns. This means that a cell would be expanding horizantally. rowspan is used to combine cells across rows. This means that a cell would be expanding vertically.

Cell spacing vs Padding: Cell spacing determines the space in between indiviual cells. A table with large cell spacing, would have gaps between the squares of each cell on all sides. Padding is the space between the content of a cell and the edge of a cell. A table with large padding would still have the cells touching but the blank space within the cell would be large.

Defining Table Borders using CSS: When styling a table to have borders in CSS, curly brackets, the term border, and the desired border size are used. An example is "table { border: 1px;}". This CSS code is then linked to the head of the html file where the table is. The width of the border is controlled by changing the numerical pixal input to a desired size.
